== Description ==
Clickpic is an classic design photography WordPress theme that comes with high-quality features. This photography WordPress Theme is ideal for any photo blogger, photographers, illustrators, graphic designers, travel, food, lifestyle, sports, and other photography website.

== Frequently Asked Questions ==

= How to customize theme ? =

Just click on Appearance -> Customize
Now you can customize all section in theme

= How to Upgrade theme to pro ?=
Just click on Appearance -> Customize 
Now find upgrade to pro button, just click on it to upgrade pro version

= How to work on Header Phone and Address =
Just click on Appearance -> Customize -> Header Phone and Address

= How to work on Header social links =
Just click on Appearance -> Customize -> ClickPic Social Link

= How to work on Banner Text? =
Just click on Appearance -> Customize -> clickpic Home Banner Text

= How to work on Footer Widgets ? =
Just click on Appearance -> Customize -> Widget

= How to work on Footer copyright ? =
Just click on Appearance -> Customize -> clickpic Footer
